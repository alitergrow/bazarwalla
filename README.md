# Bazarwalla - Web to APK System

## 🚀 How to get your installable APK (proper system)

1. **Push this repository** to GitHub (public or private).
2. Go to the **Actions** tab — a workflow named "Build APK" will start automatically.
3. After completion (approx 1-2 min), click on the run, then download **app-debug-apk** artifact.
4. Transfer the `app-debug.apk` to your Android device and install.

## 📱 App Details
- Website: https://bazarwalla.com/
- Package: com.web2apk.demo
- Theme: #1E2A3A
- Icon: auto-generated (or custom)

The whole flow is explained in the live tool: User Input → Android Project Generation → GitHub Actions → APK Artifact → Install.
